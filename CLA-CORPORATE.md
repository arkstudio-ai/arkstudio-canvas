### 1. 适用范围

本协议适用于贡献公司**任何员工、外包人员或代理人**（统称「**授权贡献者**」）以贡献公司名义或在其职务范围内向 Canvas Flow（以下简称「本项目」）提交的所有贡献。

### 2. 著作权许可

贡献公司以**永久、全球、非独占、免费、免版税、不可撤销**的方式，授予项目所有人以下权利：

(a) 复制、修改、合并、发布、再许可、销售贡献公司及其授权贡献者提交的全部贡献及其衍生作品；

(b) **以任何许可证条款（包括但不限于 AGPL-3.0、其他开源许可证、闭源商业许可证）将上述贡献及其衍生作品再次发布与再次许可给任何第三方**；

(c) 上述权利可由项目所有人自由转授，且不需事先通知或经贡献公司同意。

### 3. 专利许可

贡献公司以永久、全球、非独占、免费、免版税、不可撤销的方式，授予项目所有人及本项目用户一项专利许可，使其可以制造、使用、销售、许诺销售、进口及以其他方式转让贡献公司提交的贡献。

如果贡献公司或其任何关联方就本项目或其中包含的贡献向任何主体提起专利诉讼（包括交叉请求或反诉），则其依据本协议授予的任何专利许可在该诉讼提起之日终止。

### 4. 贡献公司的陈述与保证

贡献公司声明并保证：

(a) **授权签字人有权代表贡献公司**签署本协议并作出本协议项下的全部承诺；

(b) 贡献公司**拥有完整的著作权**（包括但不限于通过劳动合同、劳务合同、外包合同等方式合法取得的全部职务作品著作权）以授予本协议项下的许可；

(c) 授权贡献者提交的贡献是其原创作品；如包含第三方材料，该材料以与本协议兼容的方式提供，且来源与许可证已清晰标注；

(d) 该贡献不侵犯任何第三方的著作权、专利权、商标权、商业秘密或其他知识产权；

(e) 贡献公司将及时书面告知项目所有人任何可能影响上述陈述准确性的事实。

### 5. 授权贡献者名单

贡献公司应在本协议**附件 A**中列明所有被授权代表贡献公司提交贡献的人员。**只有在附件 A 中列明的人员所提交的贡献**才视为贡献公司根据本协议提交的贡献。

附件 A 可由贡献公司**单方面更新**，更新方式为：通过邮件向项目所有人发送修订后的附件 A，并由项目所有人签收确认。

### 6. 无担保 / 无义务 / 治理法律

第 5 条（无担保）、第 6 条（无义务）、第 7 条（治理法律与争议解决）、第 8 条（权利转让）、第 9 条（其他）参照《个人版贡献者许可协议》（CLA.md）相应条款，并视为本协议组成部分。

### 7. 签署方式

本协议**不通过 GitHub Pull Request 在线签署**。贡献公司须：

1. 完整填写本文件首部的公司信息表与附件 A；
2. 由授权签字人**签字并加盖公章**后扫描；
3. 将扫描件发送至项目所有人邮箱（见 README.md「商业咨询」一节公布的邮箱）；
4. 经项目所有人书面确认（邮件回复亦可）后生效。

生效后，附件 A 中列明人员的 GitHub 账号将被加入 CLA Bot 的 **allowlist**，无需在每个 Pull Request 中重复签署 ICLA。
